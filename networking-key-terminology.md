# Wired Equivalent Privacy (WEP)
security protocol for protecting wireless networks (obsolete)
# Secure Shell (SSH)
a secure network protocol for logging in and executing commands on a remote machine
# File Transfer Protocol (FTP)
a protocol for transferring files between systems on a network
# Simple Mail Transfer Protocol (SMTP)
to send and receive emails over the network
# Hypertext Transfer Protocol (HTTP)
a client-server protocol for data transfer on the Internet
# Server Message Block (SMB)
a protocol for sharing various resources (files, printers) on a network
# Network File System (NFS)
protocol for accessing files over a network
# Simple Network Management Protocol (SNMP)
a protocol for managing and monitoring network devices
# Wi-Fi Protected Access (WPA)
a wireless security protocol that uses a password to protect against unauthorized access
# Temporal Key Integrity Protocol (TKIP)
wireless network security protocol (less secure, outdated)
# Network Time Protocol (NTP)
protocol to synchronize time between computers on a network
# Virtual Local Area Network (VLAN)
a method of dividing a network into several logical networks
# VLAN Trunking Protocol (VTP)
creates and manages VLANs on multiple computers (Layer 2)
# Routing Information Protocol (RIP)
distance vector routing protocol for LAN and WAN networks
# Open Shortest Path First (OSPF)
interior Gateway Protocol (IGP) for routing traffic within an autonomous system (AS)
# Interior Gateway Routing Protocol (IGRP)
Cisco's proprietary routing protocol for autonomous systems
# Enhanced IGRP (EIGRP)
an enhanced Cisco routing protocol that combines features of distance vector and link-state protocols
# Pretty Good Privacy (PGP)
encryption and signature program
# Network News Transfer Protocol (NNTP)
a protocol for distributing and receiving messages in newsgroups
# Cisco Discovery Protocol (CDP)
a Cisco proprietary protocol for discovering and managing Cisco devices on the network
# Hot Standby Router Protocol (HSRP)
a Cisco protocol for providing router redundancy in case of failure
# Virtual Router Redundancy Protocol (VRRP)
a protocol for automatically assigning available IP routers to participating hosts
# Spanning Tree Protocol (STP)
a protocol for providing loop-free topology in Layer 2 Ethernet networks
# Terminal Access Controller Access-Control System (TACACS)
centralized authentication, authorization, and accounting protocol
# Session Initiation Protocol (SIP)
a signaling protocol for managing communication sessions (voice, video, multimedia) in IP networks
# Voice Over IP (VoIP)
allows you to make telephone calls over the Internet
# Extensible Authentication Protocol (EAP)
an authentication framework that supports passwords, certificates, and public keys
# Lightweight EAP (LEAP)
Cisco's proprietary wireless authentication protocol based on EAP
# Protected EAP (PEAP)
a security protocol that creates an encrypted tunnel for authentication in wireless networks
# Systems Management Server (SMS)
a solution for managing networks, systems, and mobile devices within an organization
# MS Baseline Security Analyzer (MBSA)
a tool from Microsoft for finding security vulnerabilities in Windows systems
# Supervisory Control and Data Acquisition (SCADA)
industrial process control and monitoring systems
# Virtual Private Network (VPN)
technology for creating a secure encrypted connection over the Internet
# Internet Protocol Security (IPsec)
a set of protocols for protecting data transmitted over IP networks
# Point-to-Point Tunneling Protocol (PPTP)
Remote Access Secure Tunneling Protocol (obsolete)
# Network Address Translation (NAT)
a technology that allows multiple devices on a private network to access the Internet through a single public IP address
# Carriage Return Line Feed (CRLF)
a combination of carriage return and line feed characters that marks the end of a line
# Asynchronous JavaScript and XML (AJAX)
a web development technique for creating dynamic pages without full page refreshes
# Internet Server Application Programming Interface (ISAPI)
an API for creating high-performance web server extensions
# Uniform Resource Identifier (URI)
syntax used to identify a resource on the Internet
# Uniform Resource Locator (URL)
a subset of a URI that includes a protocol and domain name for locating a resource
# Internet Key Exchange (IKE)
a protocol for setting up a secure connection and exchanging keys
# Generic Routing Encapsulation (GRE)
data encapsulation protocol for transmission within a tunnel
# Remote Shell (RSH)
a Unix program for executing commands on a remote computer (unsafe)
