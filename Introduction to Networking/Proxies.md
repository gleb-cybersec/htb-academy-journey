# Proxy
a device or service that is in the middle of a connection and has the ability to inspect the contents of the traffic

## Main proxy services
- **Dedicated Proxy / Forward Proxy** a request to the intermediary server and the intermediary server fulfills the request (filters outgoing requests)
- **Reverse Proxy** filters incoming requests
- **Transparent Proxy** With an opaque proxy, there must be information that it is being used. A configuration is created to ensure that traffic is directed to the proxy server. If there is no information on the server, you will not be able to connect to the network.
