## Transmission Control Protocol (TCP)
it is a connection-oriented protocol that creates a virtual channel between two devices before transmitting data using a three-way handshake

## User Datagram Protocol (UPD)
It is a connectionless protocol. It sends data packets to the recipient without checking whether they have been received

## Internet Control Message Protocol (ICMP)
used by devices to report errors and exchange information about the network status
    - **ICMP Requests** - a message sent by one device to receive information from another or to perform a specific action
    - **ICMP Messages** - to transmit various information or error information

## Voice over Internet Protocol (VoIP)
a method of transmitting voice and multimedia over the Internet
