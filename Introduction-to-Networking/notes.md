## Wide Area Network (WAN)
- the Internet is typically accessed, but it is not exclusively for the Internet
- it contains a large number of LANs
- the IP scheme does not have to comply with RFC 1918

## Local Area Network (LAN) / Wireless Local Area Network (WLAN)
- IP addresses are usually assigned from RFC 1918
- LAN wired, WLAN wireless

## Virtual Private Network (VPN)
- **Site-To-Site VPN** both the client and server are Network Devices(Routers or Firewalls)
- **Remote Access VPN** the computer creates a virtual interface that appears to be on the organization's network
  - **Split-Tunnel VPN** internet connection is not going out of the VPN
- **SSL VPN** works inside browsers
